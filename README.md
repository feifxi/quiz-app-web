# QUIZZA (A Platform Sandbox Quiz Game)

![image](https://github.com/user-attachments/assets/0ebeff86-3c4e-45fe-ac2a-2a9c3532bd7f)

## About
We are a platform where anyone can create and play quizzes with a variety of gameplay templates.
Our system supports interactive features such as comments and likes, allowing users to engage with each other. We also feature a leaderboard showcasing popular games and individual scores.
Users can freely create and customize their own quizzes, with submissions reviewed and approved by an admin to ensure quality and consistency. üòéüòé

## List Features 
- Authentication ‚Üí Authenticate system by role base. (eg. Admin, User)
    - Sign up, Sign in, Log out
- Quiz
    - Home ‚Üí A page full of playable quizzes, with comments and likes.
    - Quiz workspace ‚Üí For user, they can check quizzes' status, create, update and delete quizzes. For admin, they can approve users' and their own quizzes. They can also create, update and delete all quizzes.
    - Create quiz ‚Üí Create quizzes using various quiz's templates. (Multiple Choice, Matching, Ordering, etc.)
    - Update quiz ‚Üí Update quizzes by adding/deleting level(s), editing questions, answers and/or questions' image, and changing quiz(s)' templates.
    - Admin review quiz ‚Üí Admin can approve, view/update or delete quiz(s).
    - Comment quiz ‚Üí Can comment in the Home page.
    - Reaction ‚Üí Can press a like button for that quiz(s)
    - Play quiz ‚Üí Play quiz(s) made by various users.
    - Design quiz template type ‚Üí Quiz(s) can have various templates. (Multiple Choice, Matching, Ordering, etc.)
- Leaderboard ‚Üí Show the descending order from different categories.
    - The most star user ‚≠ê
    - Owner of the most liked quiz üëç
- Chat Bot (Optional)
